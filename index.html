Bienvenido a mi proyecto de prueba de Sass

Sass (Syntactically Awesome Style Sheets) es un preprocesador de hojas de estilo para CSS que te permite escribir código CSS de manera más eficiente y mantenible. Es una herramienta muy popular entre los desarrolladores front-end porque permite organizar y modularizar el código de una forma muy sencilla, además de agregar funcionalidades que no están disponibles en CSS nativo, como variables, funciones, mixins, bucles, entre otros.

Caracteristicas

Variables

Permiten definir variables para colores, tamaños, tipografías, etc. y reutilizarlas en diferentes partes del código.

Mixins

Permiten reutilizar bloques de código CSS en varias partes del proyecto y pasar parámetros para adaptar su comportamiento.

Funciones

Sass incluye funciones nativas para realizar operaciones como cambiar colores, calcular tamaños, etc.

Bucles

Sass permite escribir bucles en el código para repetir reglas CSS varias veces.

Anidamiento de reglas:

Permite anidar reglas CSS dentro de otras, lo que mejora la legibilidad y organización del código.
